# 🧮 Scientific Calculator

A scientific calculator built from scratch while learning Python.

## 📌 Description

This project is part of my Python learning journey. Instead of building the calculator all at once, it evolves step by step while I learn new programming concepts and software development practices.

## 🚀 Current Features

- User greeting
- Input of two numbers
- Addition
- Subtraction
- Multiplication
- Division
- Integer division
- Modulus
- Power operation

## 🛠️ Technologies

- Python 3.14
- Git
- GitHub
- Visual Studio Code
- Linux (CachyOS)

## 📈 Roadmap

- [x] Basic arithmetic operations
- [ ] Interactive menu
- [ ] Functions
- [ ] Error handling
- [ ] Scientific operations
- [ ] Calculation history
- [ ] Graphical User Interface (GUI)

## ▶️ How to Run

Create and activate a virtual environment, then run:

```bash
python main.py
```
